# Portafolio de Machine Learning y Redes Neuronales

Bienvenido a este repositorio. Aquí encontrarás una colección de prácticas y proyectos enfocados en la predicción de datos y los fundamentos de las redes neuronales artificiales.

Contenido del Repositorio

# Proyectos de Predicción
Modelos de Machine Learning aplicados a problemas de regresión y análisis de datos.

* **Predicción de la eficiencia del combustible:**
    Análisis y modelo para estimar el rendimiento de combustible en vehículos basado en sus características técnicas.
* **Predicción de popularidad de una canción:**
    Modelo diseñado para predecir qué tan popular será una canción basándose en sus atributos (tempo, género, etc.).

# Redes Neuronales y Lógica
Ejercicios prácticos sobre la arquitectura y funcionamiento básico de las neuronas artificiales.

* **Tema 4: Redes Neuronales Multicapa:**
    Implementación y estudio de redes con capas ocultas (Deep Learning).
* **McCullockPitts:**
    Ejercicios basados en la neurona de McCulloch-Pitts, uno de los primeros modelos matemáticos de una neurona biológica.
* **Compuertas Lógicas (Perceptrón):**
    * 'OPERACIÓN OR Y NOT.ipynb
    * 'OR Y NOT.ipynb'
    Notebooks que demuestran cómo una neurona puede aprender y resolver operaciones lógicas básicas (OR, NOT).
